# 日程 JSON 输出格式说明

## 根结构

这是一个 **按星期几分组的字典**。每个星期几（如 `Mon`、`Tue`）对应一个包含日期和课程信息的对象。

### 字段说明：

| 字段       | 类型     | 描述                             |
|------------|----------|----------------------------------|
| `Mon` / `Tue` / `Wed` / 等 | 对象     | 每一天的数据（按星期几分组）     |
| `date`     | 字符串   | 对应星期几的具体日期，格式为 `YYYY-MM-DD` |
| `courses`  | 数组     | 当天的课程列表                   |

### 每个课程的字段：

| 字段      | 类型     | 描述                            |
|-----------|----------|---------------------------------|
| `name`    | 字符串   | 课程名称                        |
| `room`    | 字符串   | 上课地点                        |
| `start`   | 字符串   | 课程开始时间，格式为 `HH:MM`    |
| `end`     | 字符串   | 课程结束时间，格式为 `HH:MM`    |
| `teacher` | 字符串   | 教师名称，多个教师用逗号分隔    |

---
